## Parnok-corpus-project

## Описание
Анализ корпуса поэзии Софии Парнок, состоящего из 51 текста. Проект выявляет количество слов вообще и слов уникальных, коэффициент лексического разнообразия, количество строк, длину среднего и самого длинного слова, а также самые распространенные слова в каждом тексте.

## Структура проекта
my_corpus_project/
├── corpus/              # Папка с текстовыми файлами
│   ├── text1.txt
│   ├── text2.txt
│   └── ...
├── data/                # Структурированные данные
│   └── metadata.csv
├── results/             # Результаты анализа
│   ├── statistics.csv
│   └── report.txt
├── main.py              # Главный файл с точкой входа
├── text_utils.py        # Функции анализа текста
├── file_utils.py        # Функции работы с файлами
├── README.md            # Документация проекта
└── PROJECT_REQUIREMENTS.md  # Этот файл (для справки)

## Как запустить
python main.py

results/report.txt
